# Instagram Clone
A professional full-stack clone of Instagram built with **Next.js**, **Prisma**, **Pinata** and other modern technologies. This project replicates the core features of Instagram such as authentication, image uploads, likes, comments, user profiles, and more.

## Tech Stack

- **Next.js** ‚Äî React framework for hybrid rendering (SSR/SSG).
- **React** ‚Äî UI library for building modern reactive interfaces.
- **MongoDB Atlas** ‚Äî Cloud-hosted NoSQL database used for data storage.
- **TailwindCSS** ‚Äî Utility-first CSS framework for fast styling.
- **Prisma** ‚Äî Next-gen ORM for database modeling and queries.
- **NextAuth.js** ‚Äî Secure, extensible authentication.
- **Pinata SDK** ‚Äî IPFS integration for decentralized image storage.
- **Lucide React** ‚Äî Lightweight icon library.
- **Lodash / Date-fns** ‚Äî Utility libraries for data and date manipulation.
- **React Masonry CSS / React Spinners / React Preloaders** ‚Äî Smooth, responsive UI animations and loading effects.
